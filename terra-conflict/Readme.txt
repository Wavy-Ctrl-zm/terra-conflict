TERRA CONFLICT: Global Warfare

Browser Tactical Shooter — Pure HTML/CSS/JS

TERRA CONFLICT is a fast-paced 2D tactical shooter playable entirely in your browser. Take command of elite factions, navigate diverse maps, and engage in strategic combat against enemies in dynamic environments.

Features

Multiple maps with unique environments and challenges

Selectable factions with distinct strengths

Wave-based enemy encounters and tactical gameplay

Fully built with HTML, CSS, and JavaScript — no downloads required

Gameplay Controls
Key	Action
WASD / Arrow Keys	Move your character
Mouse	Aim
Left Click / Space	Fire
R	Reload
Q	Switch Weapon
P / Esc	Pause
Getting Started

Play directly in your browser by opening the game’s GitHub Pages URL.

Compatible with modern browsers: Chrome, Firefox, Edge.

Repository Structure

At a glance, the project contains:

index.html — main entry point

css/ — styling and visual layout

js/ — core game logic and mechanics

assets/ — images, sprites, and resources

Deployment

The game can be hosted easily on GitHub Pages or any static hosting platform.

Note: This repository is for showcasing the game and its gameplay. All code and assets are original and included for demonstration purposes.
